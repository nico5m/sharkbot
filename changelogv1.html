Changelog v0.1 Testing (09/06/2024)

Aggiunto (+)
+ Aggiunto il comando /clear per pulire un canale testuale da un tot numero di messaggi (Da 1 a 99)
+ Aggiunto il comando /comandi per visionare tutti i comandi presenti nel bot
+ Aggiunto il comando /info per verificare i dati del bot come l'uptime e la latenza
+ Aggiunto il comando /kick per buttare fuori una persona dal server (Compreso di log)
+ Aggiunto il comando /msg per scrivere in privato ad un utente tramite il bot (Compreso di log)
+ Aggiunto il comando /riavvia per riavviare il bot
+ Aggiunto il comando /ssbando per mandare il bando Anticheat (FiveM)
+ Aggiunto il comando /staffbanfo per mandare il bando Staff (FiveM)
+ Aggiunto il comando /ticket per inviare il Ticket Panel (Compreso di log)
+ Aggiunto il comando /timeout per impostare in minuti un timeout ad un utente (Compreso di log)
+ Aggiunto il comando /untimeout per rimuovere un timeout ad un utente (Compreso di log)
+ Aggiunto il comando /ban per bandire un utente dal server (Compreso di log)
+ Aggiunto il comando /unban per annullare il ban ad un utente del server (Compreso di log)
